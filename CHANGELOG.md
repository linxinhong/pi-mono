# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- Initial release of pi-feishu, a Feishu (Lark) bot port of pi-mom
- Webhook-based event handling for Feishu platform
- Support for group @mentions and direct messages
- Card message support (text and code blocks)
- File attachment handling and upload
- Docker sandbox support for isolated command execution
- Event scheduling system (immediate, one-shot, periodic)
- Skills system for reusable CLI tools
- Memory persistence (global and per-channel)
- LLM context management with compaction
- Retry logic for transient failures
